# mbs-skills

> 8 AI design skills by MBS — Designer × Builder

A collection of AI-powered design skills for critiquing, auditing, and improving digital products, brands, and Web3 projects.

## Skills

- **ui-roast** — Brutal but constructive UI critique
- **website-audit** — Full website UX and design audit
- **web3-ux-audit** — Web3-specific UX audit for wallet flows and onboarding
- **landing-page-review** — Conversion-focused landing page review
- **logo-review** — Logo and brand mark critique
- **token-brand** — Token brand and identity evaluation
- **nft-art-direction** — NFT collection art direction review
- **brand-naming** — Brand naming strategy with 10 name options

## Usage

Each skill file in `skills/` defines an Assistant persona and a User prompt template. Drop the skill into any Claude-compatible agent or skill runner.
